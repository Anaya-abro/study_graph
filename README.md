# 📊 Study Tracker (Python CLI)

## 📌 Overview
This is a command-line Study Tracker application built using Python.  
It allows users to record what they studied, how long they studied, and on which day.

The data is stored both in memory and in a text file for persistence.

---

## ✨ Features
- ✅ Add study records (subject, time, day)
- ✅ View current study schedule
- ✅ Save data to a file (`study.txt`)
- ✅ Load saved study data
- ✅ Delete study entries

---

## 🛠️ Tech Stack
- Python
- File handling (`.txt` storage)

---

## ⚙️ How It Works
The program provides a menu with the following options:
1. Add study data  
2. View current session data  
3. Load saved schedule from file  
4. Delete a study record  

All added data is saved to a file for later use.
